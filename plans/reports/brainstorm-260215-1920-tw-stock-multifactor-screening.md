# 腦力激盪報告：台灣股市多因子篩選平台

**日期**: 2026-02-15
**狀態**: 已確認

---

## 問題陳述

開發一套網站針對台灣股市進行多因子篩選（籌碼+基本面+技術面），結合 LLM 解讀新聞與情緒，供小圈子投資者使用。

## 核心決策

### 技術架構

| 項目 | 決策 | 理由 |
|------|------|------|
| 前端 | Vue 3 | 用戶偏好 |
| 後端 | FastAPI (Python) | 金融數據處理 + Python 生態系強 |
| 資料庫 | MySQL | 用戶偏好，8.0+ 功能足夠 |
| 部署 | 本機/NAS | 小圈子用，零雲端成本 |
| LLM | Gemini API | 免費額度大、繁中支援好 |

### 資料來源策略

| 資料類型 | 來源 | 說明 |
|----------|------|------|
| 股價/成交量 | FinMind | 歷史+每日收盤價 |
| 三大法人 | FinMind | 外資/投信/自營商買賣超 |
| 融資融券 | FinMind | 融資融券餘額變化 |
| 月營收 | FinMind | 公司月營收年增率 |
| 季度財報 | FinMind | 損益表/資產負債表/現金流量表 |
| 技術指標 | pandas-ta 自算 | MA/KD/MACD/RSI/BB，自算更靈活 |
| 新聞 | Google News RSS | 台灣財經新聞，餵給 LLM |

**備援**: TWSE/TPEX 官方 API 作為 FinMind 不可用時的 fallback。

### 三大因子設計

#### 籌碼面 (預設權重 40%)
- 三大法人買賣超（外資/投信/自營商）
- 融資融券餘額變化
- 評分邏輯：連續買超天數、買超金額/成交量比、融資減少+融券增加=正面

#### 基本面 (預設權重 35%)
- 月營收年增率
- EPS 趨勢
- 毛利率趨勢
- ROE（股東權益報酬率）
- 負債比率
- 現金流量
- 本益比 (P/E)
- 評分邏輯：營收連續成長、EPS 上升、毛利率穩定/上升、ROE > 15%

#### 技術面 (預設權重 25%)
- 均線系統（MA5/10/20/60/120）
- KD 指標
- MACD
- RSI
- 布林通道
- 成交量變化
- 評分邏輯：均線多頭排列、KD 黃金交叉、MACD 翻正、量價配合

**權重可由使用者調整**，滑桿 UI 控制，三者總和 = 100%。

### 前置篩選條件（硬門檻）

在三大因子評分之前，先用硬條件過濾：

| 條件 | 說明 |
|------|------|
| **週成交量 > 上週 2.5 倍** | 量能爆發訊號，篩掉無量股票 |

通過硬門檻的股票才進入因子評分流程。此門檻可在設定頁調整倍數。

### LLM 分層架構

```
全市場 1700+ 支
    ↓ [硬門檻篩選：週量 > 上週 2.5x 等]
通過門檻的股票
    ↓ [Code 量化三因子評分]
Top 10-20 候選股
    ↓ [Gemini API 深度分析]
完整投資分析報告
```

**LLM 完整報告內容**:
1. 籌碼面解讀：法人動態、籌碼集中度趨勢
2. 基本面分析：財務體質、成長性評估
3. 技術面判斷：目前位階、支撐壓力
4. 新聞情緒：近期新聞摘要、情緒分析（正面/中性/負面）
5. 風險提示：潛在利空因素、需注意風險
6. 綜合建議：短中期看法

### 頁面規劃

| 頁面 | 功能 |
|------|------|
| Dashboard 總覽 | Top 股票排名、即時評分、籌碼異動提醒 |
| 個股詳情頁 | 三大因子詳細數據 + 圖表 + LLM 分析報告 |
| 自訂篩選頁 | 用戶自定義篩選條件（產業/市值/各因子門檻） |
| 籌碼統計頁 | 法人買賣超統計、融資融券統計、趨勢圖 |
| LLM 報告頁 | 所有候選股的 LLM 分析報告集中檢視 |
| 歷史/回測頁 | 歷史篩選結果、策略績效追蹤 |
| 設定頁 | 權重調整、排程設定、LLM 參數 |

### 更新機制

- 每日收盤後自動執行（~16:00）
- cron job / APScheduler 觸發
- 流程：拉數據 → 計算因子 → 評分排序 → Top N 送 LLM → 存結果

## 風險評估

| 風險 | 影響 | 緩解方案 |
|------|------|----------|
| FinMind 停止服務/限流 | 無法取得數據 | 預備 TWSE 直接抓取的 fallback |
| Gemini API 限流/費用 | LLM 分析中斷 | 控制每日只分析 Top 10-20 支 |
| 財報資料延遲 | 基本面數據不即時 | 季報本來就有延遲，可接受 |
| 技術指標計算錯誤 | 評分不準 | 單元測試 + 與外部來源交叉驗證 |

## 成功指標

- 每日自動產出 Top 10-20 股票排名
- 每支候選股有完整 LLM 分析報告
- 使用者可自行調整因子權重
- 系統在本機/NAS 穩定運行

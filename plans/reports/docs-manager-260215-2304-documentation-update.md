### 技術棧
- **後端**: FastAPI + SQLAlchemy + MySQL 8.0 + APScheduler
- **前端**: Vue 3 + TypeScript + Vite + Element Plus + ECharts
- **認證**: JWT (python-jose) + Bcrypt (passlib)
- **數據**: FinMind SDK + Google News RSS
- **LLM**: Google Generative AI (Gemini 2.0 Flash)
- **評分**: 3-因子加權 (籌碼 40% + 基本面 35% + 技術面 25%)

### 核心流程
1. **日常自動化** (16:30 執行):
   - 數據收集 (FinMind + News)
   - 硬篩選 (成交量 > 1000 萬)
   - 逐股評分 (3 個評分器)
   - 加權計算 (40:35:25)
   - 結果排名與儲存
   - AI 分析 (Gemini)
   - 報告生成

2. **使用者互動**:
   - 登入系統 (JWT)
   - 查看儀表板 (最新結果)
   - 股票詳情 (K線、評分、AI報告)
   - 自訂篩選 (靈活條件)
   - 回測驗證 (歷史績效)
